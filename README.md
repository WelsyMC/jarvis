# JARVIS

IA personnelle qui sera utilisée partout (gmail perso, domotique, planification, agenda etc...).

## Commandes importantes

- `npm run dev`: Lance le script et en watch (chaque modification du code restart le process).
- `npm run start`: Lance le script sans watch

## Important
Attention à bien remplir le .env.local, `TELEGRAM_BOT_TOKEN` et `TELEGRAM_ALLOWED_USER_ID` !
